# Stitch AI Design System Prompt

## One-Line Prompt

Design a minimalist ADHD-friendly task management web app with Coastal Calm palette (soft blue #4A90E2 primary, sage mint #7DD3C0 secondary, warm yellow #F7B731 accent), Geist font, featuring flexible tag-based organization instead of rigid boards, CRUD tasks with estimated/actual duration tracking, calendar view, reminder system, freemium model showcase (15 free tasks limit vs unlimited premium), clean dashboard with sidebar navigation, mobile-first responsive design, subtle animations under 200ms, WCAG AA accessible, inspired by Linear's smoothness and Notion's simplicity but optimized for users with ADHD who struggle with time blindness and need external brain organization without overwhelming complexity.

---

## Prompt Breakdown (For Reference)

**App Name:** Flime.ai

**Core Purpose:** ADHD-friendly task management & reminders SaaS

**Target Users:**
- People with ADHD/ADD (primary)
- Freelancers/parents juggling multiple responsibilities (secondary)

**Key Differentiators:**
- Tags instead of boards (flexible, multi-category)
- Estimated vs Actual duration tracking (combat time blindness)
- Subtasks support (break down overwhelming tasks)
- Multi-channel reminders (email, web push)

**Visual Identity:**
- **Palette:** Coastal Calm (ADHD-researched colors)
  - Primary: #4A90E2 (Soft Blue - calming, focus)
  - Secondary: #7DD3C0 (Sage Mint - tranquility)
  - Accent: #F7B731 (Warm Yellow - positivity)
  - Success: #6BCF7F, Warning: #F39C6B, Danger: #E85D75
- **Typography:** Geist (Vercel's modern UI font)
- **Style:** Minimalist, clean, distraction-free

**Core Screens to Design:**
1. Dashboard (sidebar + main content area)
2. Task creation/edit modal
3. Calendar view (monthly)
4. Settings page
5. Upgrade to Premium CTA

**Design Principles:**
- Extreme simplicity (less is more)
- No distractions (clean, calm colors)
- Fast feedback (animations < 200ms)
- Mobile-first responsive
- WCAG AA accessible

**Freemium Visual:**
- Show 15 task limit for FREE
- Clear upgrade CTAs for Premium ($5/mo)

**Inspiration:**
- Linear (smooth animations, clean UI)
- Notion (simplicity, flexibility)
- Height (minimalist task management)
